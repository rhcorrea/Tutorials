Nesse caso, o Python entenderia apenas a "pergunta" caso a caso.

Qual pergunta?

Quando usei a função isnull(), perguntei ao Python se aquele valor era nulo. Se o valor é nulo, a resposta é True; caso contrário, a resposta é False.

Voltando ao caso principal, pois esse comando nos responde muito pouco.

E agora? que fazer com esses 19.620 valores faltantes?¶

Bom, temos algumas alternativas:

A primeira delas é simplesmente descartar todas as linhas que possuam o valor NaN (Not a Number).

A segunda é "completar" os dados usando alguma técnica. Isso é chamado imputação (imputation).

Como decidir o que fazer? Largar os dados ou fazer imputation?¶

Aqui eu seguirei com imputation para demonstrar como pode ser feito.
